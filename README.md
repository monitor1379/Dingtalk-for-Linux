## Installation

Enter 'Enter' to use default installation position

```
# Step 1.
$git clone git@github.com:monitor1379/Dingtalk-for-Linux.git

# Step 2.
$cd Dingtalk-for-Linux

# Step 3.
$sudo ./install.sh
```

# Uninstallation

```
cd path/to/dingtalk
sudo ./uninstall.sh
```
